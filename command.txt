Inside the .git folder

After running:

git init

You'll have something like:

.git/
│── HEAD
│── config
│── description
│── hooks/
│── info/
│── objects/
│── refs/

Some important files/directories:

HEAD → Points to the current branch.
config → Repository-specific Git configuration.
objects/ → Stores all commits, files, and trees.
refs/ → Stores branch and tag references.
